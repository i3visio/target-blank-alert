  • Sobre Target Blank Adviser
¿Qué es eso de target="_blank"?

Se trata de una funcionalidad propia de HTML que indica al navegador que el autor de la web quiere que el enlace referido se abra en una pestaña nueva.

¿En qué consiste la vulnerabilidad?

Los principales navegadores permiten un mal uso potencial de esta funcionalidad para abrir una pestaña nueva y modificar el contenido de la original. Esta capacidad puede ser usada para ataques de phishing orientados al robo de credenciales. En esta web hay un ejemplo práctico de qué puede producir.

¿Qué puedo hacer para protegerme?

De primeras, saber que existe y que algunas de las webs que utilizas habitualmente no lo han corregido. Esta extensión te avisará indicándote el número de enlaces peligrosos que se encuentran en cada sitio web. La concienciación es el mejor camino, así que si sabes de una web que aún no lo ha corregido, repórtaselo a los adminsitradores y avisa a otros usuarios.
